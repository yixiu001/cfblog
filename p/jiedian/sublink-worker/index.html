Sublink Worker-轻量订阅转换，一键部署cloudflareWorkers，附带汉化教程

一休2024-11-292025-09-20
【YouTube频道】 | 【Telegram频道】 | 【GitHub仓库】

一、简介

Sublink Worker 是一个可部署在 Cloudflare Worker 上轻量级的订阅转换工具。它可以将各种代理协议的分享 URL 转换为不同客户端可用的订阅链接。同时还提供灵活的自定义规则与API支持。

体验地址：https://sw.yixiu920.us.kg/

二、仓库

  • 一休仓库：github仓库
    • 修改base64编码方法，修复bota编码导致的异常
  • 原作者仓库：github仓库
    • 原作者已根据一休提供的issue接受了最新代码

三、功能特点

  • 支持协议：ShadowSocks, VMess, VLESS, Hysteria2, Trojan, TUIC
  • 支持导入 Base64 的 http/https 订阅链接
  • 一键部署，Vanilla JS + Cloudflare Worker，无需后端
  • 支持客户端：
  • Sing-Box
  • Clash
  • Xray/V2Ray
    • 支持短链接生成（基于 KV）
    • 浅色/深色主题切换
    • 灵活的 API，支持脚本化操作
    • 用户友好的 Web 界面，灵活的自定义规则
    • 提供多种预定义规则集
    • 可自建关于geo-site, geo-ip, ip-cidr和domain-suffix的自定义策略组

四、部署方式

  1. fork项目
  2. 编辑README.md文件
    • 修改第40行后面的仓库地址为自己的仓库地址，然后保存
  3. 点击文档中的自动部署
  4. 根据提示获取cloudflare中的参数填入
  5. 打开工作流
  6. 等待部署完成即可食用
